# Codebase Scan

Brownfield codebase onboarding for Codex.

This plugin helps Codex build a fast, grounded model of an existing repository before planning or implementation starts.

It ships:

- `Scan Existing Codebase` for stack, architecture, conventions, integration points, and risk mapping

Use it when the repo is unfamiliar, the change is non-trivial, or you want a cleaner architecture read before deciding how to build.
